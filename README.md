# 🚀 Codic — Interactive Programming Languages Learning (demo)

https://ai-aven-ger.github.io/codic-programmer_guide/

---

## ✨ Возможности

* 🎯 Интерактивный выбор языка программирования
* 💻 Подсветка синтаксиса кода (Highlight.js)
* 📋 Копирование кода одной кнопкой
* 🌙 Тёмная / ☀️ Светлая тема
* 🔎 Поиск по языкам
* 📱 Полностью адаптивный интерфейс
* ⚡ Плавные анимации появления контента
* 🧠 Объяснения после каждого примера

---

## 🧠 Поддерживаемые языки

* 🐍 Python
* ⚡ JavaScript
* ☕ Java
* 🔧 C / C++
* 🐹 Go
* 🐘 PHP

---

## 🖼️ О проекте

Codic создан как **минималистичная обучающая среда**, которая позволяет:

* быстро изучать основы языка
* повторять синтаксис
* видеть готовые примеры
* использовать как офлайн-справочник

Всё приложение работает **полностью на клиенте** без сервера.

---

## 🛠️ Используемые технологии

* HTML5

* Vanilla JavaScript
* Highlight.js
* LocalStorage API
